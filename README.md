Este site é uma Landing Page para um evento fictício chamado "TechVerse 2026 - Festival do Futuro Digital". Esse evento reúne estudantes, desenvolvedores e 
profissionais da área de tecnologia para apresentar novas ideias, projetos e tendências do mundo digital.

As Linguagens utilizadas para esse sistema foram: HTML, CSS, PHP, JavaScript e o GitBash.

Instruções de execução: Ao abrir o site você pode visualizar a apresentação da proposta da TechVerse,
no cabeçalho do site temos as opções para: Home(parte principal do site), Sobre(texto rápido explicando sobre o que se trata o site), Atividades(Mostra a programação seguinte da TechVerse, apresentando os projetos) e Contato(Nessa parte do site fazemos contato com os desenvolvedores, enviando perguntas ou oportunidades).

Link do projeto online: https://projeto-final-uc14-rho.vercel.app/
